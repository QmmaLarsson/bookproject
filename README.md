# Projekt i kursen DT210G, Fördjupad frontend-utveckling
**Namn:** Emma Larsson\
**Student-ID:** emla2309

**Syfte med uppgift:**
Syftet med uppgiften var är att utveckla en Single Page Application med React och TypeScript där följande implementerades:
- Routing med React Router
- JWT-baserad autentisering
- Skyddade routes
- CRUD-operationer mot REST API
- Väldefinierade TypeScript-interfaces

**Lösning på uppgift:**
Applikationen är en webbapplikation där användare kan söka på böcker. Det går att visa en enskild bok för att få mer information, läsa recensioner och skriva en egen recension. För att kunna skriva, redigera och ta bort recensioner krävs det att man loggar in via JWT-autentisering.
